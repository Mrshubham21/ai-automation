# AI Automation Workflow (n8n)

## Overview

This project automates AI agency outreach using n8n.

## Features

* Lead generation (mock data)
* Lead scoring
* AI-generated personalized DMs (Groq)
* Automated follow-up messaging
* Google Sheets logging

## Tools Used

* n8n
* Groq API
* Google Sheets

## Workflow

1. Generate leads
2. Filter leads
3. Store in Google Sheets
4. Generate DM
5. Send follow-up after delay
6. Track results

## Setup

* Add Groq API key
* Connect Google Sheets OAuth
* Import workflow.json into n8n

## Note

Test accounts used. No violation of platform policies.
